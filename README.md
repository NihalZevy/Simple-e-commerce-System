# Simple-e-commerce-System
The main objective is to apply object-oriented programming (OOP) principles, proper code organization using packages, and exception handling.

To understand and implement Java packages (com.ecommerce and com.ecommerce.orders) 
To design classes using OOP concepts such as encapsulation and abstraction 
To implement a shopping cart system 
To handle errors using exception handling 
To improve code readability and structure 

 Future Enhancements :

Connect to a database 
Implement user login system
